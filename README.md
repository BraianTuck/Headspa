# Zenith HeadSpa - Versión Vanilla HTML/CSS/JS

Esta carpeta contiene la versión estática del sitio web, convertida desde la aplicación Next.js.

## Estructura
- `index.html`: El archivo principal con todo el marcado HTML.
- `styles.css`: Hoja de estilos con variables de color y animaciones personalizadas.
- `script.js`: Lógica simple para inicializar iconos y comportamiento.
- `assets/`: Carpeta con las imágenes del sitio.

## Cómo ver el sitio
Simplemente abre el archivo `index.html` en tu navegador web (doble clic). No se requiere instalación ni servidor, aunque para cargar correctamente los iconos y estilos se necesita conexión a internet (para descargar Tailwind CSS y Lucide Icons desde los CDN).
